# Juan Santiago Chronicle

An animated professor chronicle project for Juan Santiago / Stanford University.

This is a source-backed editorial chronicle documenting Juan Santiago's distinguished academic career from undergraduate studies through his election to the National Academy of Engineering.

Structure:
- Research dossier documenting academic career milestones
- Milestone timeline with comprehensive chronology
- Geographic journey map showing academic institutions
- Editorial landing page with narrative chapters
- TimelineJS + StoryMapJS integration for interactive storytelling
